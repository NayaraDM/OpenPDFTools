# 📌 Sobre o projeto

O **OpenPDFTools** é uma aplicação web desenvolvida para facilitar tarefas comuns com arquivos PDF sem necessidade de instalar programas.

A aplicação utiliza processamento local no navegador, garantindo maior privacidade, pois os arquivos do usuário não são enviados para servidores externos.


---

# ✨ Funcionalidades atuais

## 📎 Juntar PDFs

Combine vários arquivos PDF em um único documento.

Exemplo:

```
arquivo1.pdf
arquivo2.pdf
arquivo3.pdf

↓

documento_final.pdf
```


## 📑 Extrair páginas

Selecione páginas específicas de um PDF.

Exemplo:

```
Entrada:

documento.pdf

Páginas:

1,3,5


Saída:

paginas_extraidas.pdf
```


---

# 🔐 Privacidade

Todos os arquivos são processados localmente no navegador.

✅ Sem upload de documentos  
✅ Sem armazenamento externo  
✅ Mais segurança para arquivos sensíveis
